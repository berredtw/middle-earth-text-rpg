# 魔戒 · 中土遠征（純文字連線 RPG）

一款以托爾金《魔戒》為世界觀的純文字網頁遊戲，向 30 年前的 MUD 文字遊戲與《天堂》致敬。
不需安裝、不需網路也能單機遊玩；想連線時啟動小伺服器即可組隊、決鬥。

---

## 一、怎麼開始玩（單機）

1. 點兩下 **「開始遊戲.bat」**（或直接用瀏覽器打開 `index.html`）。
2. 「開始新冒險」→ 選存檔位 → 選種族 → 分配 10 點能力 → 取名字 → 踏上旅途。
3. 遊戲會自動存檔（打贏、移動時），也可隨時按左下「儲存進度」。

## 二、遊戲內容

- **種族**：人類（均衡）、精靈（技能流）、矮人（血牛）、哈比人（閃避爆擊）。
- **主線二十一章**：完整跟隨原著——夏爾 → 古墓崗 → 躍馬旅店 → 風雲頂 → 布魯南渡口 → 愛隆會議 → 卡蘭拉斯風雪 → 摩瑞亞西門水怪 → 凱薩督姆之橋（炎魔）→ 黃金森林 → 阿蒙漢（波羅莫之死）→ 法貢森林（白袍歸來）→ 伊多拉斯 → 洛汗烽火 → 聖盔谷 → 艾辛格 → 死者之道 → 帕蘭諾大會戰 → 黑門談判 → 西力斯昂哥（屍羅）→ 末日火山。每章都有開場與結尾劇情，完成後自動解鎖新地圖（中間面板右上的下拉選單）。
- **後日談「時光之門」三段外傳共三十四章＋終幕**（將魔戒投入末日火焰後解鎖）：甘道夫在瑞文戴爾開啟一道回到過去的門，親身走進魔戒的遠古歷史——完整跟隨托爾金編年史。
  - **第一段·時光之門（十一章，第二～第三紀元史詩）**：**剛多林的陷落**（炎魔之王勾斯魔格）→ **戒指工坊的淪陷** → **贈禮之主**（索倫化身「安納塔」）→ **九王的墮落**（見證九位人王化為戒靈）→ **努曼諾爾的沉沒**（黃金大帝）→ **巴拉多圍城** → **達戈拉德最後同盟**（佩戴魔戒的索倫本體）→ **格拉頓平原的伏擊**（埃西鐸之死，魔戒沉入大河）→ **安格馬的寒冬**（巫王滅北方王國——遊俠血脈的由來）→ **河畔的謀殺**（躲在蘆葦裡看史麥戈奪走魔戒、化身咕嚕的起點）。完成獲得【伊蘭迪爾之星】與稱號【時光行者】。
  - **第二段·哈比人外傳（十章，跟隨《哈比人》的冒險）**：**食人妖的篝火** → **哥布林鎮** → **黑暗中的謎語**（旁觀咕嚕與比爾博的謎語遊戲）→ **瓦爾格的圍獵**（貝奧恩救場）→ **幽暗密林的蛛網** → **長湖鎮**（巴德的黑箭）→ **孤山**（巨龍**史矛革**）→ **白議會的突襲**（把死靈法師趕出多爾哥多）→ **五軍之戰**。完成獲得稱號【五軍的英雄】。
  - **第三段·上古紀元（十三章，跟隨《精靈寶鑽》回到世界之初）**：**雙燈時代** → **奎維耶能** → **偉大旅程** → **維林諾**（神造商店）→ **雙樹的殞落**（昂哥立安）→ **星光下的戰爭** → **驟火之戰**（親歷芬國昐單挑魔苟斯的「獨舞七創」）→ **貝倫與露西安**（血口飢狼卡黑洛斯）→ **無盡的眼淚**（尼南斯戰役，屠戮者勾斯魔格）→ **屠龍者的哀歌**（龍父格勞龍與黑劍古山格）→ **憤怒之戰**（黑龍之王安卡拉剛）→ **安格班**（最終試煉：**魔苟斯**）。完成獲得【精靈寶鑽】與稱號【遠古的傳奇】。
  - **終幕·大海的呼喚**：與甘道夫同行到灰港，親手送白船啟航——第三紀元落幕，獲得稱號【第四紀元的見證者】。
  - 三段外傳共藏著五十餘件遠古傳說裝備與道具（艾格洛斯霜矛、安卡拉剛之牙、黑劍古山格、多爾露明龍盔、露西安的星影斗篷、格龍德……），以及龍鱗與虛空之絲的工房神裝配方。
- **烏塔莫深淵（無盡塔）**：上古紀元開啟後出現，**沒有層數上限**——敵人無限增強，每 10 層有樓主、每 50 層有越來越豐厚的里程碑大獎。你的深度紀錄永久保留，挑戰自己的極限吧。
- **裝備品質**：怪物掉落與工房合成的武器防具有機率帶品質——**精良**（藍字，+10%）、**卓越**（紫字，+20%）、**傳奇**（金字，+30%）。同一件裝備每次掉落的成色都不同，值得為完美的那件多刷幾次；高品質裝備賣價也更高。
- **成就系統**：背包分頁的「🏆 成就」共 28 個——從初出茅廬到放逐魔苟斯、深淵兩百層、傳奇獵人……達成時會即時通知。
- **二週目（新的輪迴）**：銷毀魔戒後，任何城鎮都有「⏳ 新的輪迴」按鈕——保留等級、裝備、金幣、夥伴、塔進度與成就，章節從頭開始，敵人每一週目生命 +60%、攻擊 +25%、防禦 +15%（經驗金幣也 +50%），可無限疊加，挑戰你能走到第幾輪。
- **首領不再是必贏的（v18 全面改版）**：每位首領都是一道「強化檢定」——**血量大幅提升**、**固定減傷**（攻擊力不足時每刀只剩個位數）、**持續回血**（輸出追不上回血就永遠磨不死）、**攻勢更重**。中後期首領還有：血量低時**灌藥水回血**、**爆擊**（1.6 倍）、瀕死**狂暴**（攻擊與攻速大增）、**大招吟唱**——卡片出現「⚠ 凝聚力量」時 3 秒後大招轟落（最多打掉你 65% 最大生命，滿血必可倖存；**暈眩技能可打斷吟唱**），以及第九章起的**詛咒技**（依原著設計）：巫王的**黑暗吐息**與格勞龍的**魔眼**使你虛弱、屍羅的蛛液**破甲**、史矛革的**龍威**使你恐懼，各持續 8 秒。戰鬥拖過 2 分半，首領**怒意**還會每 20 秒不斷升級——拖越久越危險。打不過就回頭強化、附魔、換夥伴，再來討回場子。
- **章節等級上限**：每一章有對應的等級上限（約為該章敵人等級 +6），達到上限後**經驗歸零、戰利品金幣打八折**——跟著劇情推進才能繼續成長，掛一整晚也不會跳出當前故事該有的強度。全部章節完成後解除上限。
- **懸賞板**：每個城鎮都有「📜 懸賞板」，隨機三張討伐／收集委託，完成領賞金＋正義種子×2，可付費換一批。達到等級上限後的最佳金流來源。
- **傳說的迴廊**（畢業考）：全部 56 章完成後，時光之門城鎮開放**首領連戰**——依歷史順序連續挑戰全部 32 位首領，中途不回城，每 8 連勝有里程碑賞金，32 連勝制霸獲得稱號【傳說的迴廊·制霸】。
- **圖鑑與說明**：遊戲中點選任何物品名稱（背包、裝備、商店）都會跳出完整說明與取得方式；背包分頁還有「📖 物品圖鑑」「📖 怪物圖鑑」。另附 `物品與怪物圖鑑.md` 說明檔可離線查閱。
- **至尊魔戒**：第四章會議後獲得。戰鬥中戴上可立刻脫身，但會累積「腐化」——滿 100 會被戒靈找上，下場很慘。
- **裝備強化**（天堂式）：最方便的方式——**點任何武器防具的名稱**，說明視窗裡直接有「詠唱」按鈕，成功後視窗停留，+1、+2、+3 一路點上去。也可以從卷軸那邊點「詠唱」再選目標。+3 以內安全；+4~+6 六成；**+7 起只有三成，失敗裝備直接蒸發**。祝福卷軸失敗不會壞。防具卷軸也可強化斗篷。
- **裝備取得三路線**：商店買（穩定）→ 工房合成（**略強於同級商店貨**，材料靠打怪）→ 首領掉落／劇情限定／塔頂兵器（最強）。
- **異界道具**：武士刀、妖精之弓、魔法鎧甲、加速藥水……皆為向《天堂》致敬的彩蛋裝備。
- **即時制戰鬥（v18 改版）**：戰鬥不再輪流出手——**每個角色與怪物都依自己的攻擊速度即時出招**。武器各有攻速（匕首 0.9 秒/擊最快、格龍德巨錘 1.7 秒最慢），敏捷可再加速（每點 -0.4%，上限 -30%），能力分頁與武器說明都看得到。**手動模式**：自己點「⚔ 攻擊」與技能按鈕（都有冷卻倒數）；**掛機模式**：普攻、技能、喝藥全自動。技能有各自冷卻（5~15 秒）＋共用 1 秒施法間隔；藥水有 1.5 秒冷卻防爆喝；逃跑失敗 5 秒內不能再逃。加速藥水與「王者號令」等加速效果＝**攻速提升 40%**。
- **技能**：探索會隨機遭遇 1～5 隻敵人，點敵人卡片可切換目標；各種族共 **9 招技能**，一路學到 Lv.50——Lv.8 與 Lv.26 的💥範圍技、Lv.34 的看家本領（人類「王者號令」加速／精靈大補／矮人增攻／哈比人「第二頓早餐」）、Lv.42 的高階單體絕招（必中／穿防／重暈／豪賭）、Lv.50 的終極範圍大招。爆擊率吃敏捷、爆擊傷害吃敏捷與力量（能力分頁可看）。
- **怪物屬性與特性**：怪物有六屬性——🔥火（攻擊+12%）、💧水（持續再生）、🌪風（較難命中且**攻速較快**）、⛰土（防禦+25%）、☀光（攻擊必中）、🌑暗（攻擊吸血）；部分怪物攻擊還會附帶☠中毒（持續 6 秒扣血）或💫暈眩（1.5 秒無法行動），戰鬥卡片上都有標示。
- **甘道夫的附魔**：每個城鎮都有「🧙 甘道夫的附魔」。他不收錢，只收**正義種子**（所有怪物 22% 掉落、首領必掉），5 顆換一次隨機附魔：武器五選一（劇毒／破甲／吸血／烈焰／麻痺）、防具五選一（守護／抗毒／抗暈／抗性／活力），重複附魔可覆蓋洗效果。抗毒抗暈正是對付上面那些毒怪暈怪的解答。
- **夥伴系統**：每個城鎮的「🍺 旅店招募夥伴」（800 金幣一次）可隨機招募，同時一位隨行。分低級（村民工匠 55%）／中級（遊俠、詩人、見習法師等 35%）／高級（**亞拉岡、甘道夫、勒苟拉斯等原著英雄** 10%）三檔，職能分「盾衛」（60% 替你擋刀）、「醫護」（定期治療）、「突擊」（跟著輸出）。夥伴每 2 秒行動一次，倒下回城就恢復。
- **百層挑戰塔**（仿傲慢之塔）：「摩瑞亞·無盡階梯」（第九章後）、「艾辛格·歐爾桑克塔」（第十六章後）與「多爾哥多·死靈法師之塔」（哈比人外傳，塔頂藏著死靈法師的白骨杖）。塔內八成以上會遇到 2～5 隻的怪物群（範圍技的主場），敵人隨樓層增強，**每次戰鬥勝利有 12.5% 機率發現往上的樓梯**（每 10 層的樓主打贏必定放行並掉祝福卷軸＋金幣），第 50 層有大獎，**第 100 層塔頂各藏一件獨一無二的兵器**（都靈之斧／異界·執行者之斧）。死亡不扣層數，開掛機可自動爬塔。
- **時空加速**：三級以上城鎮商店販售「異界·時空加速卷軸」（400 金），使用後 10 分鐘內**戰鬥以 2 倍速快轉、探索間隔 0.8→0.5 秒**，可疊加延長。
- **自動補給**：放置設定可勾選「自動補貨」（紅色藥水／精靈露水／加速藥水／時空加速卷軸低於 10 個自動買到 30）、「戰鬥自動喝加速藥水」與「時空加速自動續用」；所有放置設定會跟著存檔保存。自動補貨與時空續用只在「自動掛機」進行中才會花錢——死亡回村後即自動停止，不會再漏財。
- **賣出物品**：點背包的「賣」按鈕直接賣出（不再跳確認視窗），金額會顯示在日誌。**強化過的裝備賣價更高**：每級強化都會加成售價（+7 以上每級加 2200 金），高強化的多餘裝備值得拿去換錢。
- **蒐集與合成**：全部敵人都會掉落材料（狼皮、蜘蛛絲、祕銀碎片、炎魔之核……共 14 種，首領必掉稀有材料）。到任何城鎮點「🔨 工房合成」，12 張配方可打造**商店買不到的合成限定裝備**（獸牙彎刀、祕銀細劍、陰影斗篷、炎魔核心鎧、毒牙匕首等 9 件）與藥水卷軸。
- **劇情限定裝備**：西方皇族的古劍（第二章）、至尊魔戒（第六章）、精靈斗篷（第十章）、**安都瑞爾·西方之焰**（第十六章艾辛格傾覆後，聖劍重鑄）——買不到也打不到，只能跟著故事走。
- **放置掛機**：左側勾選「自動掛機」即可持續戰鬥練功——普攻與技能全自動施放，戰鬥外自然恢復 HP/MP，血量回穩自動繼續探索或爬塔，危急自動喝藥（首領戰喝藥門檻自動提高、看到大招吟唱會提前補滿），直到角色死亡才停。
- **拾獲的戰利品面板**：掉落物、品質詞綴、離線拾獲與隊伍狩獵掉落，統一顯示在冒險日誌下方的「🎒 拾獲的戰利品」面板，不再與戰鬥訊息混雜。
- **背景音樂（程式即時演奏）**：兩首原創配樂由遊戲程式即時作曲演奏（Web Audio 合成管弦——弦樂群、法國號、人聲聖歌、定音鼓與戰鼓），不需任何音樂檔案、循環完全無縫。城鎮與冒險時播放〈遠征之歌〉（D 大調 90 BPM 英雄主題），進戰鬥切換〈黑暗壓境〉（弗里吉亞調式戰鼓與銅管），戰鬥結束切回。地點面板的「🔊」按鈕或放置設定的「🎵 背景音樂」都可開關（偏好跨存檔保存）。瀏覽器限制下，第一次點擊畫面後音樂才會開始。

## 三、連線功能（組隊／決鬥）

> 連線是「受限定」的：必須輸入正確的**通關密語**才能進入大廳（預設 `mellon`，精靈語「朋友」——摩瑞亞西門的梗）。

### 最簡單：官方伺服器（免架設，24 小時在線）

- **網頁版**（https://berredtw.github.io/middle-earth-text-rpg/ ）：連線分頁已預填官方位址，直接按「連 線」即可。
- **下載版**：把位址改成 `wss://middle-earth-text-rpg.onrender.com`，密語 `mellon`。
- 伺服器閒置一段時間會沉睡，第一個連上的人若失敗，**等約一分鐘再按一次**就會醒來。

### 自己當主機（區網／私人房）
1. 點兩下 **「啟動連線伺服器.bat」**，黑色視窗保持開著（第一次啟動若 Windows 跳出「防火牆」詢問視窗，請勾選「私人網路」並按「允許存取」）。
2. 點兩下 **「查詢連線位址.bat」**，會顯示你的 IPv4 位址。
3. 告訴朋友兩件事：**位址**（`ws://你的IPv4:2941`）和**通關密語**（預設 `mellon`）。

三種連線情境：
   - **同一台電腦開兩個視窗測試**：填 `ws://localhost:2941`。
   - **同一個 Wi-Fi／區網（最簡單）**：朋友填 `ws://你的IPv4:2941`（例如 `ws://192.168.1.50:2941`）。
   - **朋友在外地（跨網路）**，三選一：
     a. 在家用分享器設定「通訊埠轉發（Port Forwarding）」把 2941 轉到你的電腦，朋友填 `ws://你的對外IP:2941`（對外 IP 可 Google「my ip」查）。
     b. 架在雲端主機上：把 `server.js` 複製上去執行 `node server.js`，並在雲端防火牆開放 2941。
     c. 使用 Tailscale 之類的虛擬區網軟體：雙方都安裝並登入同一網路後，朋友填 `ws://你的Tailscale IP:2941`（不必動分享器設定，最安全）。

### 所有玩家
1. 遊戲右側切到 **「連線」** 分頁。
2. 填入伺服器位址與通關密語 → 按「連線」，先進入**中土大廳**。
3. 之後可以：
   - **聊天**：下方輸入框（自己的發言靠右、別人的靠左）。
   - **房間**：人多時可「🚪 建立房間」（可設密碼）開自己的小天地——房間內的聊天、玩家列表、組隊、決鬥都與大廳和其他房間**完全隔離**；大廳看得到房間列表與人數，進房要密碼（若有設）。所有人離開後房間自動消失。
   - **組隊**：點其他玩家旁的「邀請組隊」，對方在聊天窗按「加入」（最多 4 人，須在同一個房間或同在大廳）。隊長走到野外地圖，會多出「🤝 隊伍狩獵」按鈕，全隊一起打怪、全員拿經驗金幣與各自的掉寶。注意：進出房間會自動離開原隊伍。
   - **決鬥**：點「決鬥」，對方按「應戰」即開打。友誼賽，輸了不損失任何東西，贏了累積決鬥勝場。

### 想改埠號或密語
`node server.js 埠號 新密語`，例如：`node server.js 2941 mypassword`。
遊戲內「通關密語」欄位要填一樣的字。

## 四、存檔

- 存檔存在瀏覽器裡（8 個存檔位）。**換瀏覽器或重灌前，請先用「匯出存檔」把存檔碼複製保存**，之後在標題畫面「匯入存檔」貼回即可。

## 五、檔案清單

| 檔案 | 用途 |
|---|---|
| `index.html` | 遊戲本體（單檔，可直接複製給朋友） |
| `server.js` | 連線伺服器（只需要 Node.js，零外掛套件） |
| `開始遊戲.bat` | 開啟遊戲 |
| `啟動連線伺服器.bat` | 啟動連線伺服器 |
| `查詢連線位址.bat` | 顯示要給朋友的連線位址 |
| `物品與怪物圖鑑.md` | 全物品／怪物／技能／章節資料（離線查閱用） |
