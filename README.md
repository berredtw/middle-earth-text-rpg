# 魔戒 · 中土遠征（純文字連線 RPG）

一款以托爾金《魔戒》為世界觀的**純文字網頁遊戲**，向 30 年前的 MUD 文字遊戲與《天堂》致敬。
單一 HTML 檔、零相依、無圖片——打開就能玩。

**▶ 立即遊玩：https://berredtw.github.io/middle-earth-text-rpg/**

## 特色

- 主線 21 章完整跟隨原著三部曲，加上三段後日談外傳（時光之門／哈比人／上古紀元）共 **40 章**
- 4 種族 × 9 技能、110 種物品、天堂式強化詠唱、甘道夫附魔、工房合成、裝備品質詞綴
- 三座百層挑戰塔＋烏塔莫無盡深淵、19 名可招募夥伴（含原著英雄）、28 個成就、二週目輪迴
- 放置掛機：智慧施法、自動喝藥補貨、離線收穫
- 至尊魔戒腐化系統：戰鬥中戴上可脫身，但腐化滿百會被戒靈找上……

## 怎麼玩

- **網頁版**：直接開上面的連結，進度存在瀏覽器（可匯出存檔碼備份）。想組隊、決鬥：右側「連線」分頁已預填**官方伺服器**，按「連 線」即進大廳（密語 `mellon`；伺服器沉睡時等約一分鐘再按一次）。
- **下載版**：下載本儲存庫，點兩下 `開始遊戲.bat`（或用瀏覽器開 `index.html`）。可連官方伺服器，也可自己執行 `啟動連線伺服器.bat`（需 Node.js）開區網私人房。詳見 [遊戲說明.md](遊戲說明.md)。存檔碼網頁版／下載版通用。
